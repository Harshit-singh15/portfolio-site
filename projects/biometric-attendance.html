Biometric-Based Attendance Management System

An end-to-end Android attendance system with session-based biometric authentication and proxy-attendance prevention.

View repository ↗

Overview

This project is an end-to-end attendance system for classroom settings. Instructors initiate a session-based attendance window, and students authenticate using their device's biometric sensor. A centralized database tracks attendance by class and subject, enabling straightforward reporting.

Approach

  • Designed a session-based attendance flow initiated by the instructor
  • Integrated device biometric authentication for student check-in
  • Built a centralized database supporting class-wise and subject-wise tracking
  • Implemented secure validation to prevent proxy attendance, with real-time sync between student devices and the instructor dashboard

Results

Login page of the Biometric Attendance System app.

The system prevents proxy attendance through device-bound biometric checks and real-time synchronization, tested across simulated classroom sessions.
