Match-Making Optimization using Linear Programming

A compatibility engine applying the Hungarian Method to a custom survey converted into a numerical trait-compatibility matrix.

View repository ↗

Overview

This project treats match-making as an assignment problem. A custom questionnaire captures personality, interests, and lifestyle traits, which are converted into a structured numerical compatibility matrix. The Hungarian Method is then applied to compute pairings that maximize overall compatibility across the group.

Approach

  • Designed a full questionnaire covering personality, interests, and lifestyle traits
  • Converted qualitative survey responses into a structured numerical compatibility matrix
  • Applied the Hungarian Method (linear programming) to compute globally optimal pairings
  • Validated results by checking pairwise compatibility scores against expected outcomes

Results

Optimized pairings generated by the linear programming approach.

The optimization consistently produces higher aggregate compatibility scores than random or greedy pairing baselines.
