Copula-Based Financial Risk Modeling Tool

A Python tool simulating joint stock-return dependencies with Gaussian, Student's t, and Vine copulas, with tail-risk diagnostics.

View repository ↗

Overview

Standard correlation measures often understate risk during extreme market events, where asset returns tend to move together more than usual. This project builds a copula-based tool to simulate joint stock-return dependencies using Gaussian, Student's t, and Vine copulas — each capturing tail dependence differently.

A web interface lets users visualize synthetic data and extreme-event simulations directly.

Approach

  • Fit marginal distributions for individual asset returns
  • Implemented Gaussian, Student's t, and Vine copula models to capture joint dependence structure
  • Built tail-dependence analysis, CVaR estimation, QQ plots, and Frobenius distance diagnostics
  • Developed a web interface to visualize synthetic data and extreme-event simulations

Results

Joint return dependencies simulated with different copula models.

The Vine and Student's t copulas captured tail dependence more accurately than the Gaussian copula, consistent with known limitations of linear correlation under stress scenarios.
